BALPRO LIFE - LUXURY REDESIGN IMPLEMENTATION GUIDE

=============================================================================
PROJECT OVERVIEW
=============================================================================

Your BalPro Life website has been completely redesigned from a traditional 
wellness brand to a premium, luxury, cinematic brand experience. All changes
maintain existing functionality while transforming the visual identity.

Design Theme: Dark luxury with gold accents
Target Audience: High-performance individuals seeking premium supplements
Brand Voice: Sophisticated, scientific, exclusive, cinematic

=============================================================================
INSTALLATION & SETUP
=============================================================================

1. No new dependencies were added - all tools already exist
2. Install dependencies (if needed): npm install
3. Start development: npm run dev

The design uses:
- React 19.2.0
- Tailwind CSS 4.1.17
- Material Symbols Outlined (from Google Fonts)
- Epilogue & Manrope fonts (from Google Fonts)

Dark Mode:

=============================================================================
WORKING WITH COMPONENTS
=============================================================================

Adding New Products:
In ProductsSection.jsx, add to products array:
```javascript
{
  id: 4,
  name: 'Product Name',
  price: '$99',
  description: 'Product description here',
  image: 'https://image-url.jpg'
}
```

Changing Colors:
1. Update in index.html Tailwind config (for global changes)
2. Or use inline styles for component-specific colors
3. All color tokens available via --color-name CSS variables

Adding Navigation Items:
In Navbar.jsx, add to navItems array:
```javascript
{ name: 'New Item', href: '#section-id' }
```

Creating New Sections:
Follow the pattern used in AboutSection.jsx:
- Use max-w-[1440px] for container
- Use px-10 for horizontal padding
- Match background colors to theme
- Use glass-morphism for cards
- Apply luxury fonts (Epilogue/Manrope)

=============================================================================
RESPONSIVE DESIGN
=============================================================================

Breakpoints:
- Mobile: < 768px (md breakpoint)
- Tablet: 768px - 1024px
- Desktop: > 1024px

Components are built mobile-first:
- Single column on mobile
- Multi-column on tablet/desktop
- Tailwind breakpoints: sm, md, lg, xl

Example:
<div className="grid grid-cols-1 md:grid-cols-3 gap-10">
  {/* 1 column mobile, 3 columns on medium+ */}
</div>

=============================================================================
ANIMATIONS & INTERACTIONS
=============================================================================

Hover Effects:
- Buttons: scale(1.05) on hover, color transition
- Cards: -translate-y-4 on hover (elevation effect)
- Images: scale(1.1) on hover
- All transitions: duration-500 for smoothness

Transitions:
- All use duration-500 (500ms) for luxury feel
- ease-out for natural deceleration

To Add More Animations:
- Use Framer Motion (already installed as dependency)
- Apply to hover states or scroll triggers
- Keep animations subtle and premium-feeling

CUSTOMIZATION GUIDE
=============================================================================

Want to Change the Luxury Theme?

1. Update Colors:
   - Edit Tailwind config in index.html
   - Change #efbf70 (gold) to new primary color
   - Change #0A0807 (obsidian) to new background

2. Update Fonts:
   - Replace Epilogue/Manrope in index.html
   - Update fontFamily settings in Tailwind config

3. Change Section Content:
   - Edit text directly in component files
   - Update product names/prices in ProductsSection
   - Modify brand story in AboutSection

4. Adjust Spacing:
   - Modify max-w-[1440px] for container width
   - Change px-10 for horizontal padding
   - Adjust py-40 for vertical padding
   - Edit gap sizes (gap-10, gap-20, etc.)

5. Update Images:
   - Replace image URLs in components
   - Use higher quality images for luxury feel
   - Ensure consistent image styling

=============================================================================
TROUBLESHOOTING
=============================================================================

Issue: Colors not appearing correctly
Solution: Clear browser cache, rebuild CSS with npm run build

Issue: Fonts not loading
Solution: Check Google Fonts CDN link in index.html

Issue: Blur effects not working
Solution: Ensure browser supports backdrop-filter, add fallback

Issue: Mobile menu not working
Solution: Check setMobileMenuOpen prop is passed correctly

Issue: Images not loading
Solution: Verify image URLs are accessible, check CDN status

Issue: Cart not working
Solution: Check onAddToCart prop is passed to ProductsSection

=============================================================================
BEST PRACTICES
=============================================================================

1. Always use the luxury fonts (Epilogue/Manrope)
2. Maintain the gold (#efbf70) as primary action color
3. Keep dark backgrounds (#0A0807, #151312) for consistency
4. Use glass-morphism sparingly for premium feel
5. Apply smooth 500ms transitions to all interactive elements
6. Test on mobile, tablet, and desktop before deploying
7. Keep aspect ratios consistent (especially for images)
8. Use max-w-[1440px] for all major containers
9. Maintain uppercase labels with letter-spacing
10. Always preview in dark mode theme
